### 一、简介

`这是一款深度支持Mybatis-Flex的IDEA插件，它最大的亮点就是可以高度自定义代码生成模板，它可以快速根据表生成代码，可以使用导出/导入把配置分享给同事以便快速开发`

### 二、宗旨

致力于简化Mybatis-Flex用户操作繁琐且毫无意义的额外操作
